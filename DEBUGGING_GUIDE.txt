ISSUE 1: "Unique INSTITUTIONS: [...]" doesn't include your faculty
SOLUTION:
  - Copy the exact institution name from console
  - Your logged-in user's faculty doesn't match this value
  - UPDATE: currentUser.faculty needs to match database institution

ISSUE 2: "Unique DEPARTMENTS: [...]" doesn't include your department  
SOLUTION:
  - Copy the exact department name from console
  - Your logged-in user's department doesn't match this value
  - UPDATE: currentUser.department needs to match database department

ISSUE 3: All strategies fail but "Final result: X records"
SOLUTION:
  - Check if X is > 0
  - If yes, scholars ARE being loaded (use fallback strategy)
  - If no, check database connectivity and permissions
